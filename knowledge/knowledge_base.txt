1. НАЗНАЧЕНИЕ РОТАЦИИ ОФИЦИАНТОВ В РЕСТОРАНЕ (на телефоне)
----------------------------------------------------------------------

Шаг 1. Откройте «Меню»
  - На главном экране приложения внизу есть три вкладки: «Главная», «Уведомления», «Меню».
  - Нажмите на вкладку «Меню» (иконка из четырёх квадратов, справа внизу).

Шаг 2. Выберите «Инструменты»
  - В открывшемся меню вы увидите список разделов: Касса, Отчёты, Сотрудники, Аналитика, Работа с клиентами, Моя компания, Инструменты.
  - Нажмите на пункт «Инструменты».

Шаг 3. Выберите «Столики»
  - В разделе «Инструменты» появятся пункты: Столики, Подарки, Привлечение, WhatsApp-рассылки.
  - Нажмите на «Столики» (описание: Назначение, открепление, просмотр, настройка).

Шаг 4. Выберите ресторан
  - Откроется экран «Столики» со списком ваших заведений и филиалов.
  - Нажмите на нужный филиал (например, выделен красной рамкой).

Шаг 5. Нажмите «Назначить»
  - На экране «Действия» есть три варианта: Назначить, Открепить, Просмотр.
  - Нажмите кнопку «Назначить».

Шаг 6. Выберите зал
  - Откроется экран «Залы» со списком залов и количеством столов в каждом.
  - Выберите нужный зал (например, «Зал Рахмет6 — 10 столов»).

Шаг 7. Выберите столики
  - На экране «Назначение официанта» отображаются номера столов в виде кружков с цифрами.
  - Нажмите на номера столов, которые хотите назначить официанту (выбранные столы подсветятся зелёным/синим цветом).
  - Внизу экрана появится красная кнопка «Далее — X стола» (где X — количество выбранных столов).
  - Нажмите эту кнопку.

Шаг 8. Назначьте официанта
  - Появится список официантов. Выберите нужного (например, отметьте кружком слева от имени).
  - Внизу экрана нажмите красную кнопку «Назначить 1 официанта».

Шаг 9. Подтвердите назначение
  - На экране подтверждения вы увидите сводку: Зал, Столы, Официанты.
  - Внизу две кнопки: «Отмена» и красная кнопка «Подтвердить».
  - Нажмите «Подтвердить».

Шаг 10. Готово
  - Появится сообщение с зелёной галочкой: «[Имя] успешно назначен на столы».
  - Внизу две кнопки: «К просмотру» и красная кнопка «Готово».
  - Нажмите «Готово».

Шаг 11. Закройте окно
  - Появится всплывающее сообщение «Назначение прошло успешно».
  - Нажмите кнопку «Закрыть».

2. КАК СДЕЛАТЬ ВОЗВРАТ? (на телефоне)
----------------------------------------------------------------------

Шаг 1. Откройте «Меню»
  - На главном экране внизу нажмите вкладку «Меню» (иконка из четырёх квадратов).

Шаг 2. Выберите «Касса»
  - В списке меню нажмите на пункт «Касса».

Шаг 3. Выберите «Предзаказ»
  - В разделе «Касса» вы увидите пункты: Принять оплату по QR, История платежей, Предзаказ, Активация купонов, Онлайн Ресторан.
  - Нажмите «Предзаказ» (описание: Управление статусами предзаказов).

Шаг 4. Выберите терминал
  - Откроется экран «Предзаказ» со списком филиалов. Каждый филиал может иметь несколько типов: delivery, express, takeaway.
  - Выберите нужный терминал в зависимости от типа заказа.

Шаг 5. Откройте детали заказа
  - На экране со списком заказов вы увидите заказы с именами клиентов, статусами (например, «Заказ завершён») и номерами заказов.
  - Найдите нужный заказ и нажмите ссылку «Открыть детали заказа» (синий текст под номером заказа).

Шаг 6. Нажмите «Сделать возврат»
  - В деталях заказа вы увидите: состав заказа (блюда, количество, цены), информацию о клиенте (Имя, Номер, Сегмент).
  - Внизу в разделе «Действия» нажмите на ссылку «Сделать возврат» (выделена красной рамкой).

Шаг 7. Выберите вариант возврата
  - На экране «Отмена заказа» два варианта:
    * «Отменить частично» — Отменить заказ одного или нескольких блюд. Сумма за отменённое блюдо вернётся к гостю.
    * «Отменить полностью» — Заказ будет полностью отменён. Вся сумма заказа вернётся к гостю.
  - Выберите нужный вариант и нажмите синюю кнопку «Далее».

Шаг 8. Выберите позиции для удаления (при частичном возврате)
  - На экране «Частичная отмена заказа» отметьте галочками блюда, которые нужно отменить.
  - Внизу отображается итоговая сумма.
  - Нажмите красную кнопку «Удалить позиции».

3. КАК ВЫДАТЬ ДОСТУП СОТРУДНИКУ? (на телефоне)
----------------------------------------------------------------------

Шаг 1. Откройте «Меню»
  - Внизу экрана нажмите вкладку «Меню».

Шаг 2. Выберите «Сотрудники»
  - В списке меню нажмите на «Сотрудники».

Шаг 3. Нажмите «Добавить сотрудника»
  - На экране «Сотрудники» внизу есть синяя кнопка «Добавить сотрудника».
  - Нажмите на неё.

Шаг 4. Выберите роль и нажмите «Перейти к выбору сотрудников»
  - Откроется экран «Выберите роль» с вариантами:
    * «Администратор» — Принятие заказов, Оборот, Отзывы, Добавление сотрудников, Возврат, Назначение официантов, Управление меню и стоп-листами, Редактирование графика работы и информации о заведении.
    * «Бариста/Кассир» — Принятие заказов, Оборот, Стоп-листы, Чаевые.
    * «Официант» — Принятие заказов, Назначение за столиками, Чаевые, Активация купонов.
  - Выберите нужную роль (она подсветится).
  - Нажмите синюю кнопку «Перейти к выбору сотрудников».

Шаг 5. Укажите номер сотрудника
  - В поле поиска введите номер телефона сотрудника (формат: +7...).
  - Система найдёт сотрудника, после чего нажмите «Добавить».

4. КАК ПРИНЯТЬ ЗАКАЗ ВРУЧНУЮ? (на телефоне)
----------------------------------------------------------------------

Шаг 1. Откройте «Меню»
  - Внизу экрана нажмите вкладку «Меню».

Шаг 2. Выберите «Касса»
  - В меню нажмите «Касса».

Шаг 3. Выберите «Предзаказ»
  - Нажмите на «Предзаказ» (описание: Управление статусами предзаказов).

Шаг 4. Выберите филиал
  - На экране «Предзаказ» вверху выберите «Локация».
  - Ниже появится список филиалов по типам: delivery, takeaway, express.
  - Выберите нужный филиал.

Шаг 5. Найдите нужный заказ
  - Заказы можно найти по разделам: «Новый», «Готовится», «Готов», «Завершён».
  - Также можно воспользоваться поиском (иконка лупы вверху справа) и найти по номеру заказа.

Шаг 6. Откройте детали заказа
  - Нажмите на нужный заказ, затем нажмите ссылку «Открыть детали заказа».
  - В деталях увидите состав заказа (блюда, кол-во, цены), пожелания, информацию о клиенте.

Шаг 7. Пробейте заказ в кассе
  - Пробейте заказ вручную в вашей кассовой системе.
  - Выберите способ оплаты «Choco».
  - Вернитесь в приложение Choco Бизнес.

Шаг 8. Примите заказ
  - В открывшемся окне выберите примерное время готовки.
  - Нажмите кнопку «Принять заказ».

Шаг 9. Подтвердите готовность
  - Когда заказ будет готов, откройте его в разделе «Готовится».
  - Нажмите зелёную кнопку «Заказ готов».

Шаг 10. Выдайте заказ
  - Когда заказ выдан гостю, нажмите зелёную кнопку «Заказ выдан».

5. КАК ЗАКРЫТЬ ПРИЁМ ЗАКАЗОВ? (на телефоне)
----------------------------------------------------------------------

Шаг 1. Откройте «Меню» → «Касса» → «Предзаказ»
  - Повторите шаги 1–3 из инструкции №4.

Шаг 4. Выберите терминал
  - Выберите нужный терминал в зависимости от типа заказа (delivery / express / takeaway).

Шаг 5. Откройте «Настройки»
  - На экране предзаказа вверху справа нажмите иконку шестерёнки «Настройки».

Шаг 6. Выберите «Расписание»
  - В меню настроек выберите пункт «Расписание».

Шаг 7. Нажмите «Закрыть постоянно»
  - На экране «График работы предзаказа» в разделе «Выбор режима» вы увидите варианты:
    * «Открыто по графику» — Предзаказ закрывается за 30 минут. Открытие и закрытие согласно графику работы филиала.
    * «Открыть заведение»
    * «Закрыть до конца смены» — Предзаказ будет недоступен до конца смены. Автоматически откроется в первый и рабочий день работы согласно графику работы заведения.
    * «Закрыто постоянно» — Предзаказ не будет доступен гостям. Чтобы включить, выберите режим «Открыто по графику» или «Открыть 24/7».
    * «Закрыто по техническим причинам»
  - Нажмите «Закрыто постоянно».
  - Внизу нажмите красную кнопку «Применить».

Шаг 8. Выберите причину закрытия
  - На экране «Укажите причину закрытия» выберите из списка: «Нет ингредиентов», «Кухня не работает», «Нет света».
  - Или нажмите «Свой ответ» и напишите причину.
  - Нажмите синюю кнопку «Выбрать».

Шаг 9. Нажмите «Применить»
  - Подтвердите закрытие, нажав красную кнопку «Применить».

6. КАК ОТКРЫТЬ ПРИЁМ ЗАКАЗОВ? (на телефоне)
----------------------------------------------------------------------

Шаг 1. Откройте «Меню» → «Касса» → «Предзаказ»
  - Повторите шаги 1–3 из инструкции №4.

Шаг 4. Выберите терминал
  - Выберите нужный терминал.

Шаг 5. Откройте «Настройки»
  - Нажмите иконку шестерёнки «Настройки» вверху справа.

Шаг 6. Выберите «Расписание»
  - Нажмите «Расписание».

Шаг 7. Нажмите «Открыто по графику»
  - На экране «График работы предзаказа» выберите режим «Открыто по графику».
  - Описание: Предзаказ закрывается за 30 минут. Открытие и закрытие согласно графику работы филиала.

Шаг 8. Выберите нужный вариант и нажмите «Выбрать»
  - На экране «Закрывать предзаказ за...» выберите время:
    * за 20 минут
    * за 30 минут
    * за 45 минут
    * за 1 час
  - Выберите вариант «по графику работы филиала».
  - Нажмите синюю кнопку «Выбрать».

Шаг 9. Нажмите «Применить»
  - Подтвердите открытие, нажав красную кнопку «Применить».

7. КАК ВРУЧНУЮ ЗАКРЫТЬ НА ТИП ОПЛАТЫ ЧОКО (если оплата не упала в кассу — оплатили через QR)
----------------------------------------------------------------------

Эта инструкция выполняется на кассовом планшете/терминале (не в мобильном приложении).

Шаг 1. Выберите нужный столик
  - На кассовом экране слева отображается список залов и столов.
  - Нажмите на нужный столик (выбранный столик подсветится жёлтым).

Шаг 2. Выберите количество гостей и пробейте заказ вручную
  - Укажите количество гостей.
  - Начните добавлять позиции заказа.

Шаг 3. Добавьте блюда, которые заказал гость
  - Блюда можно посмотреть в заказе Choco Business.
  - В правой части экрана отображаются категории меню. Выберите нужную категорию и добавьте блюда.
  - Внизу будет отображаться итоговая сумма (например, «2 993 тг»).

Шаг 4. Нажмите на пречек
  - Внизу кассового экрана есть панель с иконками.
  - Нажмите на иконку пречека (значок документа/чека).

Шаг 5. Завершите оплату через Choco SR
  - В окне оплаты выберите «Банковские карты».
  - Затем выберите «Тип оплаты» → «Choco SR».
  - Введите точную сумму заказа (используя цифровую клавиатуру на экране).
  - Нажмите красную кнопку «ОПЛАТИТЬ» в правом нижнем углу.
