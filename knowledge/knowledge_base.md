# База знаний Choco Smart Restaurant

> Структурированная инструкция для сотрудников ресторана: официантов, кассиров и администраторов.
> Действия выполняются в мобильном приложении **Choco Бизнес** или в **веб-версии** (rahmet.biz). Платформа указана в каждом разделе.

---

## 1. Заказы

### 1.1 Как найти и проверить заказ

**Платформа:** Мобильное приложение
**Путь:** Меню → Касса → Предзаказ

1. Нажмите вкладку **«Меню»** внизу экрана (иконка из четырёх квадратов).
2. Выберите **«Касса»** из списка разделов.
3. Нажмите **«Предзаказ»** (описание: Управление статусами предзаказов).
4. Выберите нужный терминал в зависимости от типа заказа (delivery / express / takeaway).
5. Заказы разделены по статусам: **Новый → Готовится → Готов → Завершён**.
6. Нажмите на заказ, затем **«Открыть детали заказа»** (синий текст).
7. В деталях видно: состав, кол-во, цены, пожелания, данные клиента.

---

### 1.2 Как принять заказ вручную (если не попал в кассу)

**Платформа:** Мобильное приложение
**Путь:** Меню → Касса → Предзаказ

1. Откройте **Предзаказ** и выберите терминал.
2. Непринятый заказ будет в разделе **«Новый»**.
3. Нажмите на заказ → **«Открыть детали заказа»**.
4. Вручную пробейте заказ на кассе, выберите способ оплаты **«Choco»**.
5. Вернитесь в приложение, выберите время готовки.
6. Нажмите **«Принять заказ»**.
7. Когда готово → нажмите зелёную кнопку **«Заказ готов»**.
8. После выдачи → нажмите **«Заказ выдан»**.

**Важно:** Если заказ пришёл через Choco, но чек на кассе не вышел — всегда принимайте вручную!

---

### 1.3 Как закрыть приём заказов

**Платформа:** Мобильное приложение
**Путь:** Меню → Касса → Предзаказ → Настройки → Расписание

1. Откройте **Предзаказ** и выберите терминал.
2. Нажмите иконку **шестерёнки** (Настройки) вверху справа.
3. Выберите **«Расписание»**.
4. На экране «График работы предзаказа» выберите **«Закрыто постоянно»**.
5. Нажмите красную кнопку **«Применить»**.
6. Выберите причину закрытия: «Нет ингредиентов», «Кухня не работает», «Нет света» или **«Свой ответ»**.
7. Нажмите **«Выбрать»**, затем **«Применить»**.

**Режимы работы:**
- **Открыто по графику** — предзаказ закрывается за 30 мин до конца графика филиала. Открывается и закрывается автоматически.
- **Открыть заведение** — заведение открыто для предзаказа сейчас.
- **Закрыть до конца смены** — предзаказ будет недоступен до конца смены, откроется автоматически на следующий рабочий день.
- **Закрыто постоянно** — предзаказ полностью отключён. Чтобы включить, выберите «Открыто по графику» или «Открыть заведение».
- **Закрыто по техническим причинам** — временная остановка.

### 1.4 Как открыть приём заказов

**Платформа:** Мобильное приложение
**Путь:** Меню → Касса → Предзаказ → Настройки → Расписание

1. Откройте **Предзаказ** → Настройки → Расписание.
2. Выберите режим **«Открыто по графику»**.
3. Предзаказ закрывается за 30 минут до конца графика.
4. Укажите, за сколько минут закрывать предзаказ: 20 / 30 / 45 мин / 1 час.
5. Нажмите **«Выбрать»**, затем **«Применить»**.

## 2. Оплата

### 2.1 Как закрыть на тип оплаты Choco (оплата через QR не попала в кассу)

**Платформа:** Кассовый планшет/терминал (не в мобильном приложении)

1. На кассовом экране слева выберите нужный **столик** (подсветится жёлтым).
2. Укажите количество гостей.
3. Добавьте блюда из заказа (состав смотрите в Choco Бизнес).
4. Нажмите на иконку **пречека** (значок документа внизу экрана).
5. Выберите **«Банковские карты»** → **«Тип оплаты»** → **«Choco SR»**.
6. Введите точную сумму заказа.
7. Нажмите красную кнопку **«ОПЛАТИТЬ»**.

---

### 2.2 Как посмотреть историю платежей (мобильное)

**Платформа:** Мобильное приложение
**Путь:** Меню → Касса → История платежей

1. Нажмите **«Меню»** внизу экрана.
2. Выберите **«Касса»**.
3. Нажмите **«История платежей»** (все платежи, возвраты, статистика).
4. Нажмите **«Фильтр»** вверху экрана.
5. Выберите период (Сегодня / другой) и тип операции (Все операции / Оплата / Возврат).
6. Нажмите **«Применить»**.
7. Увидите оборот в локации за выбранный период.

---

### 2.3 Как посмотреть историю платежей (веб-версия)

**Платформа:** Веб-версия (rahmet.biz)
**Путь:** Касса

1. В боковом меню нажмите **«Касса»**.
2. Используйте фильтры вверху:
   - **«Все заказы»** — фильтр по типу заказа.
   - **«Все филиалы»** — выберите нужный филиал и нажмите **«Применить»**.
   - **«Все операции»** — выберите тип: Все операции / Оплата / Возврат.
   - **«Сегодня»** — выберите период (Сегодня / Вчера / За неделю / Последние 30 дней / За отчётный месяц / За отчётный год / За всё время / Свои даты) и нажмите **«Выбрать»**.
3. На экране отображаются: **Покупки**, **Возвраты**, **Сумма возвратов**, **Оборот**.
4. Ниже — таблица **«Транзакции за сегодня»** с деталями каждой транзакции.

## 3. Возвраты

### 3.1 Как сделать возврат

**Платформа:** Мобильное приложение
**Путь:** Меню → Касса → Предзаказ → Детали заказа

1. Откройте **Предзаказ**, выберите терминал.
2. Найдите нужный заказ → **«Открыть детали заказа»**.
3. Внизу в разделе «Действия» нажмите **«Сделать возврат»**.
4. Выберите вариант:
   - **Отменить частично** — выберите конкретные блюда для отмены.
   - **Отменить полностью** — вся сумма вернётся гостю.
5. Нажмите **«Далее»**.
6. При частичном возврате — отметьте галочками блюда, нажмите **«Удалить позиции»**.

**Критически важно:**
- Сумма возврата автоматически рассчитывается с учётом обслуживания.
- Возврат **всегда** делается в двух местах: Choco Бизнес **+** кассовая система.

## 4. Отчёты и аналитика

### 4.1 Как посмотреть отчёты по каналам оплаты (веб-версия)

**Платформа:** Веб-версия (rahmet.biz)
**Путь:** Отчёты → Канал оплаты

1. В боковом меню нажмите **«Отчёты»**.
2. Перейдите на вкладку **«Канал оплаты»**.
3. В таблице каналов оплаты выберите **«Choco Smart Restaurants»** (также доступны: Choco Life 2.0, Choco Delivery).
4. Появится окно оплаты Choco Smart Restaurants с данными по каждому платежу:
   - Номер заказа, дата, сумма заказа, комиссия, кешбэк, скидка, к выплате.
   - Способ оплаты: Choco Pay / Kaspi и др.
5. Можно нажать **«Выгрузить отчёт»** для скачивания в Excel.

**Данные на главной странице отчётов:**
- Сумма заказов, сумма возвратов, комиссия, кешбэк, скидка
- Выплата от Choco
- Начислено бонусов, списано бонусов
- Количество заказов, количество возвратов

---

### 4.2 Как посмотреть отчёты по каналам оплаты (мобильное)

**Платформа:** Мобильное приложение
**Путь:** Меню → Отчёты

1. Нажмите **«Меню»** внизу экрана.
2. Выберите **«Отчёты»**.
3. Нажмите **«Все филиалы»** — выберите нужный филиал и нажмите **«Применить все»**.
4. Выберите период (Последние 30 дней / другой).
5. Нажмите **«Канал оплаты»** внизу экрана.
6. Выберите нужный канал оплаты (Choco Smart Restaurants / Choco Life 2.0 / Choco Delivery).
7. Увидите данные: транзакции, комиссия, кешбэк, скидка, к выплате.
8. Можно нажать **«Выгрузить отчёт в Excel»**.

---

### 4.3 Как сделать сверку отчётов и закрыть кассу

**Путь:** Меню → Отчёты

1. Проверьте сумму заказов.
2. Если есть программа лояльности/скидки, используйте формулу:
   **Сумма заказов + Списано бонусов + Скидки = Итоговая сумма для сверки**
3. Сравните с отчётом кассовой системы.

**Важно:** При расхождениях — сразу решайте вопросы!

5. Нажмите **«Перейти к выбору сотрудников»**.
6. Введите номер телефона сотрудника (формат: +7...).

## 7. Управление меню

### 7.1 Как изменить блюдо в меню (веб-версия)

**Платформа:** Веб-версия (rahmet.biz)
**Путь:** Инструменты → Меню

1. В боковом меню нажмите **«Инструменты»**.
2. Выберите **«Меню»**.
3. В разделе «Ваши меню» найдите нужное меню и нажмите **«Подробнее»**.
4. В списке **«Категории»** найдите нужную категорию и нажмите **«Открыть блюда»**.
5. У нужного блюда нажмите **«Изменить»**.
6. На экране «Изменение блюда» можно:
   - Изменить **фотографию** (кнопка «Изменить» / «Удалить», мин. 256x256px, макс. 2 МБ).
   - Редактировать **название** на трёх языках (RU / KZ / EN).
   - Изменить **описание** на русском.
   - Указать **цену** и **время приготовления** (в минутах).
   - Добавить **группу дополнений** (соусы, напитки, сиропы).
7. Нажмите красную кнопку **«Сохранить»**.

**Доступно в разделе Инструменты:**
- Меню — управление блюдами и категориями
- Стоп-лист — временное отключение блюд
- Сезонные продукты
- Столики — назначение официантов
- Приложение — настройки приложения
- Интеграции — подключения к кассовым системам
- WhatsApp рассылки — маркетинговые рассылки
- Скидка на доставку

## 9. Даяшылар/кассирлер үшін нұсқаулық (Қазақ тілінде)

> Осы бөлім ресторан қызметкерлеріне арналған: даяшылар, кассирлер және әкімшілер.
> Барлық әрекеттер **Choco Бизнес** мобильді қосымшасында орындалады.

---

### 9.1 Мейрамханадағы даяшыларды ротациялау (телефон арқылы)

**Платформа:** Мобильді қосымша
**Жол:** Мәзір → Құралдар → Үстелдер

1. **«Мәзір»** батырмасын басыңыз (экранның астындағы төрт шаршы белгіше).
2. **«Құралдар»** бөлімін таңдаңыз.
3. **«Үстелдер»** дегенді басыңыз (тағайындау, ажырату, қарау, баптау).
4. Мейрамхананы таңдаңыз.
5. **«Тағайындау»** батырмасын басыңыз.
6. Қажетті залды таңдаңыз.

## 15. Автоматическое назначение официанта (iiko)

> Как работает автоматическое назначение официанта при поступлении заказа через Choco в системе iiko.

---

### 15.1 Процесс автоматического назначения

**Платформа:** iiko терминал (касса)

Когда поступает новый заказ от Choco для зала — появляется официант с именем **«Пользователь iiko»**. Все заказы с Choco сперва попадают на официанта-распределителя **«Пользователь iiko»** и затем автоматически падают на другого, уже реального официанта. Обычно этот процесс занимает **30–60 секунд**.

Заказ автоматически перейдёт к реальному официанту и заказ исчезнет у официанта **«Пользователь iiko»**.

Иногда автоматическое назначение официанта может происходить дольше и может занимать **2–3 минуты**.

---

### 15.2 Важные правила

**Платформа:** iiko терминал (касса)

**ВАЖНО!**
Когда вам поступил заказ от **«Пользователь iiko»** — **нельзя** его открывать, редактировать или отправлять на кухню.

Нужно подождать пока заказ автоматически упадёт на официанта:
- Заказ распределится на официанта, **закреплённого за столом**, откуда был сделан заказ.
- Если никто не закреплён за столом — заказ упадёт на **любого официанта**, у кого открыта смена.

---

### 15.3 Что делать при сбое?

**Платформа:** iiko терминал (касса)

**ЕСЛИ** прошло более **3 минут** и заказ всё ещё висит на официанте **«Пользователь iiko»** — значит в функционале/интернете произошёл **сбой**.

В таком случае:
- Откройте данный заказ и **назначьте официанта на данный заказ вручную**.

## 16. Продукты Choco Smart Restaurant

### 16.1 Заказ без официанта

Гость сканирует QR-код на столе, сам выбирает блюда из электронного меню и оплачивает онлайн. Заказ автоматически падает в кассовую систему (iiko / Paloma / R-keeper).

**Преимущества:**
- Персонал разгружен от приёма заказов — фокусируется на сервисе
- Средний чек растёт на 15-20% за счёт upsell-подсказок в электронном меню
- Ошибки в заказах сведены к нулю — гость сам выбирает
- Скорость обслуживания выше — не нужно ждать официанта

**Подходит для:** ресторанов, кафе, баров с посадочными местами

---

### 16.2 Заказ без кассира

Гость у стойки сканирует QR и заказывает самостоятельно. Кассир не нужен.

**Преимущества:**
- Нет очередей — каждый гость заказывает через свой телефон
- Экономия на персонале — можно работать без кассира
- Конверсия выше — гость не уходит из-за очереди

**Подходит для:** кофейни, фастфуд, фудкорты

---

### 16.3 Автоподтягивание счета (закрытие через QR)

Официант работает как обычно: принимает заказ, вбивает в iiko. Когда гость готов уйти — сканирует QR на столе и видит свой счёт. Оплачивает онлайн, не дожидаясь терминала.

**Преимущества:**
- Оборот столов ускоряется на 15-20% — гость не ждёт терминал
- Официант не бегает с терминалом — больше времени на сервис
- Для заведений, где важен живой контакт официанта с гостем

**Важное отличие от «Заказ без официанта»:**
- «Заказ без официанта» = гость САМ выбирает и заказывает через QR
- «Автоподтягивание» = официант ВБИЛ заказ в iiko, гость только ОПЛАТИЛ через QR

**Подходит для:** рестораны с полным обслуживанием, где важен контакт официанта

---

### 16.4 Киоск самообслуживания

Заказ через терминал самообслуживания. Гость выбирает блюда на экране и оплачивает.

**Подходит для:** фастфуд, кофейни, столовые

---

### 16.5 Интеграции с кассовыми системами

Choco интегрируется с основными кассовыми системами Казахстана:
- **iiko** — полная интеграция: заказы, оплаты, меню, стоп-лист, скидки, API
- **Paloma** — интеграция заказов и оплат
- **R-keeper** — интеграция заказов и оплат

Все заказы автоматически падают в кассу — нет ручного ввода, нет ошибок, экономия времени.

---

### 16.6 Веб-версия (rahmet.biz)

Веб-платформа для управления заведением с компьютера:
- Касса и история платежей
- Отчёты по каналам оплаты
- Работа с клиентами и отзывами
- Управление сотрудниками
- Управление меню и стоп-листом
- Собственная система доставки (без комиссий маркетплейсов)

**Важно:** в суперприложении Choco доставки нет, но веб-версия позволяет организовать собственную доставку как мощный канал продаж.
